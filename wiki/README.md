# SEMAS LLM Wiki

> A Karpathy-style local wiki for absorbing, tracing, and evolving ideas around
> self-evolving agent systems.

## Purpose

This directory is the **thinking/ideation log** for the SEMAS framework. While
`OPERATION_LOG.md` records what was done, the wiki records **why** — the ideas,
papers, GitHub repos, and design decisions that shaped the framework, plus
their citation sources.

## Conventions

- Each note is a markdown file.
- Every claim or borrowed idea is tagged with `[source: ...]`.
- Use short, atomic notes. Prefer bullets over prose.
- Cross-link between notes with relative paths.
- When an idea is absorbed into the codebase, note the file path and commit-ish
  if applicable.

## Index

- [`semas_evolution_ideas.md`](./semas_evolution_ideas.md) — Core SEMAS design
  and all external self-evolution methodologies we have considered or absorbed.
- [`references.md`](./references.md) — Centralized BibTeX-like reference list.
